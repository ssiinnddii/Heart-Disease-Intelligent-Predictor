Machine Learning Pipeline

The model is a scikit-learn Pipeline that chains preprocessing and classification into a single serialisable object. Using a Pipeline ensures the exact same transformations applied during training are automatically applied at inference time — preventing data leakage and keeping training and serving consistent.

1 Numeric Preprocessing
Median imputation for missing values → Standard scaling (mean=0, std=1)
2 Categorical Preprocessing
Mode imputation → Ordinal encoding (safe for tree-based models)
3 Gradient Boosting
100 estimators, random_state=42. Best F1 across 7 tested algorithms.

Feature Glossary

The model uses 13 clinical features to assess heart disease risk. These are the same measurements a doctor would review during a cardiac evaluation.

Feature Description Type
age Age in years Numeric
sex Biological sex (Male / Female) Categorical
cp Chest pain type — asymptomatic, typical angina, atypical angina, non-anginal Categorical
trestbps Resting blood pressure (mm Hg) Numeric
chol Serum cholesterol (mg/dl) Numeric
fbs Fasting blood sugar > 120 mg/dl (True / False) Categorical

thalch Maximum heart rate achieved (bpm) Numeric
exang Exercise-induced angina (True / False) Categorical
oldpeak ST depression induced by exercise relative to rest Numeric
